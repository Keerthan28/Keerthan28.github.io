# Keerthan Nekkanti — Portfolio

A One Piece pirate-themed personal portfolio for Keerthan Nekkanti.
**Live site:** [keerthan28.github.io](https://keerthan28.github.io/)

## What's inside

- `index.html` — the main portfolio (3D Thousand Sunny, wanted poster, voyage log, treasury, Haki & Devil Fruits, contact)
- `post-agi.html` — *The Search for AGI · Hunting the All Blue*
- `post-sensory-ai.html` — *Sensory Devices & AI · Engineering Observation Haki*
- `post-agentic-ai.html` — *From Bank Vaults to Agentic Crews*

## Stack

- Pure HTML / CSS / vanilla JS — no build step
- [Three.js](https://threejs.org/) (CDN, r128) for the animated Thousand Sunny in the hero
- Google Fonts: Pirata One, Cinzel, IM Fell English, Special Elite, JetBrains Mono
- Inline SVG symbol library: Jolly Roger, Straw Hat, Compass, Devil Fruit, Den Den Mushi, Anchor

## Local preview

Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000/
```
